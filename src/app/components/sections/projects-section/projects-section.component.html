Projects

HRApp is a web application that aims to help with human resources administration and recording the working hours of company employees.

The application is built from scratch and mainly consists of a MySQL database, a backend implemented with Spring and the application developed with Angular.

SnapMapp is an Android application that consists of a social network where you can share images about the place you are with your friends.

It is a social network that offers alternatives to the infinite scrolling of other applications by offering an interactive map that positively influences the user's mind.
